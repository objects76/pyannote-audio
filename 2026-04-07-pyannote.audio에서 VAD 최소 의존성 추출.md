## 수정된 최소 파일 구성

### 변경 사항

| 파일 | 이전 | 변경 | 이유 |
| --- | --- | --- | --- |
| `utils/hf_hub.py` | 유지 | **완전 제거** | 다운로드 불필요 |
| `core/model.py` | 유지 | **유지 + 대폭 단순화** | HF 다운로드 분기 제거 |
| `core/pipeline.py` | 유지 | **유지 + 단순화** | `from_pretrained()` 제거 |

### 사용 패턴 (로컬 전용)

```python
from pyannote.audio import Model
from pyannote.audio.pipelines import VoiceActivityDetection

# HF 다운로드 없이 로컬 경로 직접 지정
model = Model.from_pretrained("/weights/pytorch_model.bin")

pipeline = VoiceActivityDetection(segmentation=model)
pipeline.instantiate({
    "onset": 0.5,
    "offset": 0.5,
    "min_duration_on": 0.0,
    "min_duration_off": 0.0,
})

vad = pipeline("audio.wav")
```

`Pipeline.from_pretrained()` 을 **전혀 호출하지 않으므로** `core/pipeline.py` 의 HF 관련 로직도 모두 제거 가능합니다.

### 추가 제거 가능 여부 확인

**`check_dependencies()` 도 제거 가능한가?**  
로컬 파일이므로 버전 불일치 걱정이 없다면 아래처럼 stub으로 대체:

```python
def check_dependencies(metadata):
    pass  # 로컬 weights 사용 시 버전 체크 불필요
```

**`models/segmentation/PyanNet.py` 제거 가능한가?**  
`segmentation-3.0` 체크포인트만 사용한다면 `SSeRiouSS.py` 만 필요합니다. 다운받은 `pytorch_model.bin` 의 메타데이터를 확인해서 어떤 클래스를 참조하는지 확인하면 확실해집니다:

```python
import torch
ckpt = torch.load("pytorch_model.bin", map_location="cpu")
print(ckpt.get("pyannote.audio", {}))  # 클래스 이름 확인
```

**`core/pipeline.py`** — `pyannote.pipeline.Pipeline` → 단순 Python 클래스

```python
# ❌ 제거
from pyannote.pipeline import Pipeline as BasePipeline
class Pipeline(BasePipeline):
    # 하이퍼파라미터 최적화 관련 전부
    def get_metric(self): ...
    def parameters(self): ...  # optuna 연동
    def apply(self, trial): ...

# ✅ 변경 후 — 필요한 것만
class Pipeline:
    def instantiate(self, parameters: dict):
        """하이퍼파라미터 직접 설정"""
        for name, value in parameters.items():
            setattr(self, name, value)

    def to(self, device):
        for model in self._models.values():
            model.to(device)
        for inference in self._inferences.values():
            inference.to(device)
        return self

    def __call__(self, file):
        raise NotImplementedError
```
